Learn You a Python is an online resource created with the aim of helping anyone with any skill level to learn the programming language Python in a project-based approach. The progression is going to follow a roadmap-like structure. People should feel free to start at any checkpoint and jump around if they feel comfotrable. Topics are going to be presented and explained in an application that actually performs something that might be useful and you will rarely encounter foo bar biz . This is still a work in progress and has not been proofread yet, feedback and suggestions are highly encouraged. You can reach me using email.

This work is not meant to be a standalone resource. The best way to learn any programming language (and probably anything in life) is to tap into multiple resources and try to absorb as much as you can.
